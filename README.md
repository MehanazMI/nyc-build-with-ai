## Project: StageSense — The AI That Sees What You Can't

An AI-powered live presentation coach built on Google ADK and Gemini Live API.
Real-time coaching for speakers AND audience engagement analysis — no text boxes, just pure multimodal AI.

- **Coach Mode**: Phone faces speaker → AI analyzes pace, clarity, energy, filler words → live scorecard
- **Room Read Mode**: Phone faces audience → AI reads engagement, confusion, excitement → alert whispers

**Tech Stack**: Gemini Live API · Google ADK · FastAPI · WebSocket · SSE · Cloud Run
