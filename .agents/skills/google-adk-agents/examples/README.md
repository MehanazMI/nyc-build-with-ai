# google-adk-agents skill examples

This directory contains reference implementations for each major ADK pattern.
Drawn from the Way Back Home hackathon project (Levels 0–5).

## Examples

| File | Pattern | Level |
|------|---------|-------|
| `before_agent_callback.py` | Pre-flight state setup | L1 |
| `parallel_agent.py` | Concurrent specialists | L1 |
| `mcp_custom_server.py` | FastMCP server on Cloud Run | L1 |
| `a2a_remote_agent.py` | RemoteA2aAgent A2A client | L4 |
| `kafka_a2a_bridge.py` | Kafka request-reply shim | L5 |
| `gemini_live_websocket.py` | Gemini Live + WebSocket | L3 |
| `sse_streaming.py` | Server-Sent Events | L5 |
