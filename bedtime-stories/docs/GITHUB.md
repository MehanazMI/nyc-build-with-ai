# 🚀 GitHub Setup

## Create GitHub Repository

1. **Go to GitHub:** https://github.com/new

2. **Repository settings:**
   - **Repository name:** `bedtime-stories-ai`
   - **Description:** `AI-powered bedtime story generator using Google Gemini - transforms photos into personalized stories`
   - **Visibility:** Public (for hackathon demo)
   - **DO NOT initialize with README, .gitignore, or license** (we already have these)

3. **Create repository**

---

## Connect Local Repository

After creating the GitHub repository, run these commands:

```bash
# Add remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/bedtime-stories-ai.git

# Push to GitHub
git push -u origin master
```

---

## Quick Commands

```bash
# View current remote
git remote -v

# Change remote if needed
git remote set-url origin https://github.com/YOUR_USERNAME/bedtime-stories-ai.git

# Push changes
git push
```
