# Hello Gemini Example

This is your first agent! It tests basic Gemini API connectivity.

## Setup

1. Get your Gemini API key from [Google AI Studio](https://aistudio.google.com/)
2. Copy `.env.example` to `.env` in the root directory
3. Add your API key to `.env`

## Run

```bash
cd examples/hello-gemini
python agent.py
```

## What it does

- Connects to Gemini 2.0 Flash model
- Sends a test prompt
- Displays the response
- Confirms your API is working
