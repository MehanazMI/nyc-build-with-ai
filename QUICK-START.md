## 🎯 NEXT IMMEDIATE STEPS

### Step 1: Get Your Gemini API Key (5 minutes)

1. Open: https://aistudio.google.com/
2. Sign in with Google account
3. Click "Get API Key" → "Create API Key"
4. Copy the key

### Step 2: Add API Key to .env

1. Open the `.env` file in this directory
2. Replace `your-gemini-api-key-here` with your actual key:
   ```
   GEMINI_API_KEY=AIzaSy...your-actual-key...
   ```
3. Save the file

### Step 3: Test Your Setup

Run in PowerShell:
```powershell
python check-setup.py
```

You should see: "🎉 SUCCESS! You're ready to build!"

### Step 4: Run Your First Agent

```powershell
cd examples\hello-gemini
python agent.py
```

This will send a test prompt to Gemini and display the response!

## 🆘 TROUBLESHOOTING

### API Key Issues
```powershell
# Test if key is loaded
python -c "from dotenv import load_dotenv; import os; load_dotenv(); print('Key found' if os.getenv('GEMINI_API_KEY') else 'Key missing')"
```

### Import Errors
```powershell
# Reinstall packages
pip install --upgrade google-genai python-dotenv
```

### PowerShell Permission Errors
```powershell
# Run as administrator if needed
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
