## 🎯 Choose Your Track

### Option 1: Live Agent 🎤
Build real-time, voice-and-vision enabled agents that can converse naturally and handle interruptions.

**Examples:**
- Real-time translator with voice I/O
- Vision-enabled tutor with speech
- Accessibility assistant
- AR tour guide with audio

**Key Features:**
- [ ] Voice input/output
- [ ] Real-time responses (< 500ms latency)
- [ ] Interrupt handling
- [ ] Visual/audio feedback

### Option 2: Creative Storyteller ✨
Build agents that generate rich, interleaved outputs—seamlessly mixing text, audio, and visuals.

**Examples:**
- Photo story generator
- Interactive audiobook creator
- Visual poetry generator
- Multi-modal travel guide

**Key Features:**
- [ ] Text generation
- [ ] Audio synthesis
- [ ] Image generation/analysis
- [ ] Cohesive narrative flow

## 💡 Tips for Success

1. **Start Small, Iterate Fast**
   - Get a basic prototype working first
   - Add features incrementally
   - Test early and often

2. **Focus on the Experience**
   - Don't just demo features
   - Show a compelling use case
   - Make it interactive

3. **Leverage Multimodal**
   - Combine voice, vision, and text
   - Create seamless interactions
   - Think beyond chatbots

4. **Prepare Your Demo**
   - 3-5 minute presentation
   - Show, don't just tell
   - Have backup plan for technical issues
